Open Source at Limeleaf

We love open-source software and use it instead of commercial products whenever possible.

Limeleaf and our member-owners are involved in a number of open-source projects.

As Maintainers

  • Limecast, a privacy-first podcasting platform.
  • Libera.Codes, a free, privacy-first, QR code generator.
  • static-httpd, an HTTP static file server written in Hare.
  • pprof.sh, user-friendly CLI utility for running Benchmark*() Go tests.
  • Various Go modules, a collection of Go modules
  • Various Hare modules, a collection of Hare modules

As Contributors

  • fast_afpacket, for raw socket networking with support for Linux's AF_PACKET and SO_TIMESTAMPING.
  • firewall_toolkit, a high level wrapper for programmatically managing nftables' expressions, rules, and BPF filters.
  • Go Programming Language, an easy to learn programming language built for concurrency, networking, and performance.
  • Hare Programming Language, a systems programming language designed to be simple, stable, and robust.
  • Helix Editor, a post-modern text editor written in Rust.
  • Starship, a cross-platform, minimal, blazing-fast, and infinitely customizable prompt.
